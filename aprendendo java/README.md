## 💡 Dicas de Migração Python → Java

| Aspecto | Python | Java |
|---------|--------|------|
| **Tipagem** | Dinâmica | Estática |
| **Indentação** | Obrigatória | Opcional (usa `{}`) |
| **Ponto e vírgula** | Não usa | Obrigatório `;` |
| **Declaração** | `x = 10` | `int x = 10;` |
| **Print** | `print()` | `System.out.println()` |
| **Comentários** | `#` | `//` ou `/* */` |
| **Listas** | `list` | `ArrayList` ou arrays |
| **Funções** | `def` | métodos com tipo |

## 📌 Convenções de Código Java

- **Classes**: PascalCase (`MinhaClasse`)
- **Métodos/Variáveis**: camelCase (`minhaVariavel`)
- **Constantes**: UPPER_SNAKE_CASE (`MINHA_CONSTANTE`)
- **Pacotes**: lowercase (`com.exemplo`)
